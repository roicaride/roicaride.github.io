Proyectos

Predicción de Ventas con ML

Modelo de Machine Learning para pronóstico de ventas.

Proyecto enfocado en el sector retail, donde se desarrolló un modelo de regresión para predecir ventas mensuales con altos niveles de precisión. Se utilizaron Python, scikit-learn y técnicas de forecasting para ayudar a la empresa a optimizar su inventario.

Monitor IoT para Agricultura

Sistema IoT para monitorear condiciones en tiempo real.

Proyecto de IoT que integra sensores de humedad y temperatura conectados a una plataforma en la nube. Desarrollé tanto el hardware con microcontroladores Arduino como la plataforma de análisis de datos para brindar a los agricultores información en tiempo real y alertas tempranas. Incluye un panel web desarrollado con JavaScript para visualizar datos históricos.

Análisis Financiero Automatizado

Dashboard de análisis de datos financieros con automatización.

Implementación de un dashboard interactivo para el análisis de indicadores financieros de una empresa. Utilicé Python (Pandas, Matplotlib) para automatizar la recopilación y limpieza de datos, y una interfaz web con HTML/CSS/JS para visualizar tendencias, comparativos y proyecciones, facilitando la toma de decisiones estratégicas.
